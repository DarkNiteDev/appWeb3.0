# App Web3.0

Complete web3 application,
with transactions for sending and receiving cryptocurrencies from the Ethereum network, 
through Metamask wallet integration.
Front end(ReactJS), 
Back end(HardHat) 
Solidity Smart Contract.
Use of GIPHY API.
